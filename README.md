A very small mod that adds the ability for players to add text above a player's name (like a note about them (saved in a .txt file so the text above the player's name won't be lost after a restart))

<img width="1082" height="540" alt="image" src="https://github.com/user-attachments/assets/1f790e45-8d12-4048-bfd3-f99a98ce3238" />


The mod opens its own menu by pressing F6.

To create colored notes, you need the mod: https://github.com/Antropomeda/ColorEnabler-Crab-Game-Mod
